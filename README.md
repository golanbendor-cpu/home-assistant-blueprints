## 🏠 **System Overview**
- **12 rooms** with automated lighting (presence/motion sensors)
- **Manual override protection** (1.5min timer system)
- **Sleep mode** and **Partner day override** global controls
- **Night dimming curves** with smooth transitions
- **TV automation** with multiple wake-on-LAN attempts

## 🔒 **Security**
- **Private Git repository** for automation files (contains device IDs)
- **Public templates** in `public_gist_templates/` (safe for sharing)
- **GitHub Desktop ready** with `.gitignore` protection

## 🎯 **Daily Workflow**
1. **Make changes** to automation files
2. **Test in Home Assistant** 
3. **Update documentation** (`bugs.md`, `helpers.md`, etc.)
4. **Commit changes** using `GIT_WORKFLOW.md`
